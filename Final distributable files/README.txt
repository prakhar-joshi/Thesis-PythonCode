To run the python file in windows for the first time:
1. Save the python file and the requirements.txt file in the same folder on your PC
2. open command prompt 
3. change directory to the location of your folder thorugh: cd /d "full address of your folder"
4. Type: pip install -r requirements.txt
5. Wait for packages to be installed, this might take a minute and is a one time thing
6. Type: python WT_GReAT.py
